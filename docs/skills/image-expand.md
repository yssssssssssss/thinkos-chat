# 图片尺寸扩展

## 概述

将图片扩展到指定尺寸，支持多种填充模式。

## 支持的填充模式

| 模式 | 说明 | 示例 |
|------|------|------|
| fill | 填满整个区域，可能裁剪图片边缘 | 背景图、产品图 |
| fit | 保持比例，周围留白 | 头像、证件照 |
| stretch | 拉伸填满，可能变形 | 特殊比例要求 |

## 预设尺寸

| 预设 | 尺寸 | 用途 |
|------|------|------|
| HD | 1280×720 | 高清视频 |
| 全高清 (FHD) | 1920×1080 | 桌面壁纸、视频 |
| 2K QHD | 2560×1440 | 高分屏 |
| 4K UHD | 3840×2160 | 超高清打印 |
| Instagram | 1080×1080 | Instagram 正方形 |
| A4 海报 | 2480×3508 | 打印海报 |
| 小红书 | 1080×1920 | 竖版封面 |
| 封面图 | 1200×630 | 社交分享 |

## 使用方法

1. 点击输入区域右侧的"更多"按钮
2. 选择"图片扩展"
3. 选择预设尺寸或自定义尺寸
4. 选择填充模式
5. 点击"执行扩展"

## 技术细节

- 输出格式：PNG
- 最大支持：8192×8192 像素
- 处理方式：前端 Canvas 处理
