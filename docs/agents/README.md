# Agent 说明

## 什么是 Agent

Agent 是一个智能助手，能够理解用户的自然语言请求，并自动选择合适的技能（Skill）来执行任务。

## 工作流程

1. **意图识别**：分析用户输入，判断用户想要做什么
2. **技能选择**：根据意图选择合适的技能
3. **参数提取**：从用户输入中提取必要参数
4. **执行技能**：调用对应的技能处理任务
5. **返回结果**：将处理结果返回给用户

## 示例对话

```
用户：把图片扩展到 1920x1080
Agent：好的，我将把图片扩展到 1920×1080，使用填充模式。
✅ 已完成！
```

## 支持的指令模式

### 图片扩展

| 模式 | 示例 |
|------|------|
| 基础扩展 | "把图片扩展到 1920x1080" |
| 预设尺寸 | "扩展到 FHD" |
| 带模式 | "用填充模式扩展到 4K" |
